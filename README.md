# Advanced Number Guessing Game

A console-based Python game that challenges the player to guess a secret number with varying difficulty levels. 
The game features unique constraints per difficulty level, a proximity hint system, scoring, and multi-round play.

## --Features-- ##

- Three difficulty levels with distinct number sets:
  - Easy: prime odd numbers from 1 to 50 (5 attempts)
  - Medium: Prime numbers from 1 to 40 (5 attempts)
  - Hard: Integers from 1 to 30 (5 attempts)
- Proximity hints indicating how close the guess is (Ice cold, Cold, Warm, Hot, Very hot).
- Scoring system that rewards fewer attempts and higher difficulty.
- Multi-round gameplay.
- Input validation with user-friendly prompts.
- Ability to replay and select different difficulty levels or quit.

## How to Play ##
- Choose a difficulty level or quit.
- For each round, guess the secret number based on the valid number set.
- Receive hints after each guess.
- You have limited attempts per round.
- After rounds, see your total score.
- Option to play again with a different difficulty or quit.
